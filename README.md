# Fontilizer
Javascript Framework
by Julian Ismael Berger

## Install Guide

Include JS file in HTML file

End of BODY tag
<script src="[PATH_TO_FILE]/fontilizer.js"></script>

Inside HEADER tag
<script defer src="[PATH_TO_FILE]/fontilizer.js"></script>
